# temporal-graph-extraction
Проект извлекает ключевые события, участников и временные метки, формируя на их основе связный граф. Использует предобученную модель RuT5 для суммаризации и кастомный детерминированный алгоритм для генерации обучающего датасета с учетом контекстных связей (без явных союзов).
